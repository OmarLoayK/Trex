# 🦖 T-Rex Runner Game

A simple browser game inspired by the Chrome Dino game.

## 🎮 Play the Game

👉 https://fishswag.github.io/Trex/

## 🚀 Features

* Endless runner gameplay
* Score tracking
* Obstacles and increasing difficulty

## 🕹 Controls

* Press **Space** to jump
* Click restart after losing

## 🛠 Built With

* JavaScript
* p5.js
* p5.play
